Silly projects I've done



Most of them involve emoji. The rest of them probably involve cats. All of them live on GitHub.

  • muster: a website that lets you play with overlapping patterns
  • is-ie10-dead: a most exciting countdown, now over
  • emoji-translate: an app to translate english to emoji
  • caturday-post: cat haiku push notifications
  • meow-to: a client side URL shortener
  • emoji-rain: a Polymer element that makes it rain emoji. Like the confetti button to the right
  • twemojify: a Chrome extension that uses Twitter's twemoji library to fix Chrome's emoji
  • cat-dns: a DNS server that resolves everything to cats
  • dear sir or madam: a bookmarklet that transforms web pages into ransom notes
  • ruby-eh: a bit of ruby that makes your code more Canadian
  • diagramaphone: a website that lets you code diagrams, because drawing them really sucks
